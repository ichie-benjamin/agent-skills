Scene layout archetypes

The ten reusable scene layouts. Don't invent cold — pick the one that fits the beat, then adapt its treatment. Each demo shows how the layout sits in frame; the build adds the motion.

01

Hero Statement

A single sentence carries the whole beat — openings, turns, the closing punch.

Layout
Centered, big negative space above.
Motion
Per-word reveal, breath-hold on the hero word.
Color
Ink on bg; one accent on the pivot word only.

02

Term / Definition Card

A term must land before the argument uses it.

Layout
Label above · term hero (serif) · definition beneath.
Motion
Term sets first, definition fades up after a beat.
Color
Neutral; accent only if the term IS the concept.

03

Contrast Split

A comparison or tension between two things.

Layout
Split with a seam; each side labeled.
Motion
Sides enter from opposite edges, meet at the seam.
Color
One side neutral, the other the tension accent.

04

Enumerated Reveal

A short list (3–5) where the count itself matters.

Layout
Left-aligned list on a baseline grid, index numerals.
Motion
Items reveal one-by-one in rhythm with narration.
Color
Neutral list; accent the one emphasized item.

05

System Diagram

How parts connect or data moves — pipeline, loop, architecture.

Layout
Nodes on a grid, thin connectors, one focal node.
Motion
A dot flows along the connectors (= the data moving).
Color
Nodes neutral; the key path accent.

06

Code / Terminal Focus

A concrete, checkable example.

Layout
Mono block, dim chrome, one highlighted line.
Motion
Focus pull — block dims, the key line brightens.
Color
Mono on near-bg; accent the proving line.

07

Number Punch

One number is the whole point.

Layout
Giant numeral centered, tiny caption beneath.
Motion
Count-up to the value (finite) or a hard scale-in snap.
Color
Ink numeral; accent if it marks the tension.

08

Book-Quote Card

The canonical, verified quote.

Layout
Quote hero (serif italic), attribution small beneath.
Motion
Quote fades up as one block; attribution sets last.
Color
Ink, no accent — restraint signals authority.

09

Signature-Device Moment

The beat that embodies the episode's core concept. Exactly one.

Layout
Defined by the device itself — pull from the spec.
Motion
The peak of the energy arc; room before and after.
Color
The device's own; no competing hero motion near it.
10

Breather / Transition

Connective tissue — after a peak, before a shift.

Layout
Near-empty — bg, grain, maybe one small mark.
Motion
Ambient only (slow bloom, drift) or a clean match-cut.
Color
Bg + texture; no accent.
